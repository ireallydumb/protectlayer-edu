## Concepts

### Hardware Fingerprinting
Creating unique identifiers for devices based on:
- CPU, GPU, memory info
- Network card MAC address
- Hard drive serial number
- Device UUID
- BIOS/UEFI information

### Device Attestation
Verifying that devices are legitimate and secure:
- Check for jailbreak/root
- Verify security patches
- Confirm trusted OS
- Validate app integrity

### Geo-Restrictions
Limiting access based on geographic location:
- IP geolocation
- Regional licensing
- Time-based availability
- VPN/proxy detection
